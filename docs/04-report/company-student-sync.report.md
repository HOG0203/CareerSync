# PDCA Report: 업체정보 - 학생 취업 현황 자동 연동

## 1. 개요 (Overview)
업체정보 페이지의 '취업생 현황'을 학생 취업 데이터베이스와 실시간 연동하여 데이터 정합성을 확보하고 관리 편의성을 개선하였습니다.

## 2. 주요 변경 사항 (Key Changes)
### 2.1 서버 액션 수정 (`src/app/(dashboard)/company-info/actions.ts`)
- **데이터 조회 로직 정상화**: `student_employments` 테이블의 `id`가 `students` 테이블의 `id`와 동일한 구조임을 반영하여 조인 로직을 수정하였습니다.
- **필터링 조건 강화**: 취업처(`company`) 명칭뿐만 아니라 취업 상태(`employment_status = '취업'`) 조건을 추가하여 실제 취업자만 표시되도록 개선하였습니다.
- **불필요한 컬럼 제거**: 존재하지 않는 `student_id` 컬럼 호출을 제거하고 필요한 데이터만 선택적으로 조회하도록 최적화하였습니다.

### 2.2 디자인 문서 업데이트
- `docs/02-design/features/company-info.design.md`에 최신 데이터 연동 전략을 반영하였습니다.

## 3. 검증 결과 (Validation)
- [x] 특정 업체 선택 시 해당 업체에 '취업' 상태인 학생만 목록에 노출됨을 확인.
- [x] 학생 관리 페이지에서 취업 상태나 업체를 변경할 경우, 업체정보 페이지에 즉시 반영됨을 확인.
- [x] 현장실습생 현황('진행중', '채용전환') 탭의 데이터 연동 정상 동작 확인.
