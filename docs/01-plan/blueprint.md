# **App Name**: CareerSync

## Core Features:

- Student Profile & Employment Entry: A centralized interface for administrative staff to input and manage student profiles and their detailed employment history, including company, role, industry, and start date.
- Interactive Employment Dashboard: Visualize key employment metrics and trends through interactive charts and graphs, showcasing overall employment rates, average salaries by major or year, and industry distribution at a glance.
- Advanced Filtering & Reporting: Empower users to filter employment data by various criteria such as graduation year, major, company type, or employment status, and generate customizable reports for analysis.
- AI-Powered Trend Analysis Tool: Utilize a generative AI tool to identify and highlight significant employment trends, predict future job market outlooks for specific fields, and provide a textual summary of key factors influencing student placement.
- User Authentication & Access Control: Secure login functionality with distinct roles (e.g., admin, staff) to ensure data security and manage access levels to sensitive employment information.
- CSV Data Import/Export: Facilitate efficient data management by allowing bulk upload of existing student and employment records via CSV/Excel files and export reports in various standard formats.

## Style Guidelines:

- Primary color: A sophisticated and professional deep blue (#22A0DB), chosen to convey reliability and clarity, ideal for displaying crucial employment data. (HSL: 210, 70%, 45%)
- Background color: A very light, subtly cool-tinted grey (#E7EFF2) provides a clean and expansive canvas for data visualization, ensuring excellent readability. (HSL: 210, 20%, 95%)
- Accent color: A vibrant aqua-green (#80E0E0) to highlight interactive elements, calls to action, and important data points on the dashboard, creating dynamic contrast. (HSL: 180, 60%, 70%)
- Body and headline font: 'Inter', a modern sans-serif, for its excellent readability across different scales and its clean, objective appearance suitable for data-heavy applications.
- Use clean, minimalist vector icons that clearly represent data management functions, charts, filtering, and reporting, maintaining a professional aesthetic.
- Employ a structured, grid-based layout for dashboards, prioritizing clear data segmentation and easy navigation, with distinct sections for input forms and data visualizations.
- Incorporate subtle, smooth transitions for data loading, chart updates, and dashboard section changes to enhance user experience without distraction.
