# [PLAN] 지능형 학년 필터 기반 맞춤형 대시보드 시스템

## 1. 개요
대시보드 진입 시 사용자의 역할(Role)과 담당 학급 정보에 따라 최적화된 학년 대시보드를 자동으로 보여주는 시스템을 구축합니다. 3학년은 기존 취업 지표를 유지하고, 1·2학년은 진로 준비 지표 중심으로 새롭게 구성합니다.

## 2. 지능형 초기값 설정 규칙 (Smart Default)
- **관리자(Admin) & 비담임**: 3학년 대시보드 기본값.
- **담임 교사**: 본인 담당 학년(`assigned_grade`) 대시보드 우선 표시.

## 3. 학년별 대시보드 레이아웃 구성

### 3.1 3학년 View (현행 취업 실전 중심)
- 취업률 요약 카드, 기업 유형별 분포, 전공별 취업 현황 등 기존 모든 차트 유지.

### 3.2 1·2학년 View (신규 진로 탐색 중심)
1. **기초진로희망 차트**: 학년별 맞춤 옵션(대/공기업, 아우스빌둥, 가업승계 등) 분포.
2. **자격증 취득 현황**: 학생별 자격증 보유 수 및 학과별 평균 취득 지표.
3. **병역 희망 차트**: 군특성화, 부사관, 군대입소 등 병역 계획 분포.

## 4. 데이터 연동 전략 (학반 관리 데이터 직결)
- **Source**: `students` 테이블의 실시간 데이터 사용.
- **Fields**: `career_aspiration`(진로희망), `certificates`(자격증), `military_status`(병역희망).
- **Real-time**: 담임교사가 학반 관리 탭에서 수정 시 대시보드 즉시 반영.

## 5. 기술적 구현 계획
- **UI**: `DashboardFilters`에 학년(1, 2, 3) 선택 필터 추가 (전체 항목 제외).
- **Logic**: 선택된 학년에 따라 3학년용 컨테이너와 1·2학년용 컨테이너를 조건부 렌더링.
- **Query**: `src/lib/data.ts`에 학년별 필터링이 적용된 그룹 집계 함수 신설.

## 6. 단계별 로드맵
1. **[Plan]**: 1·2학년 전용 3대 지표 및 지능형 필터 전략 승인.
2. **[Design]**: 1·2학년용 3대 차트(진로희망, 자격증, 병역)의 시각화 형태(Pie/Bar) 및 배치 설계.
3. **[Do]**: 학년 필터 구현, 사용자 프로필 연동, 1·2학년 전용 대시보드 뷰 개발.
4. **[Check]**: 학년별 데이터 정확성 및 필터 작동 여부 검증.
5. **[Act]**: 최종 완료 보고.
