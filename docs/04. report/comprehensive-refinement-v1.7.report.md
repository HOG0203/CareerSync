# 시스템 고도화 및 대시보드 최적화 보고서 (v1.7)

## 📝 1. 개요
본 작업에서는 프로젝트 브랜딩을 **'대공커리어싱크'**로 전환하고, 학생 관리와 학급 관리 페이지 간의 항목을 동기화했습니다. 또한 모바일 UI의 사용성을 개선하고, 대시보드에 세부 교육 경로를 분석할 수 있는 '희망 진로 코스' 차트를 추가했습니다.

## 🚀 2. 주요 변경 사항

### 2.1 브랜딩 및 용어 통일
- **브랜딩 변경**: 로고, 메타데이터, 문서 내 명칭을 'CareerSync'에서 **'대공커리어싱크'**로 변경했습니다.
- **용어 표준화**: '특이사항', '진로코스 및 특이사항' 등의 명칭을 **'진로 코스'**로 통일하여 직관성을 높였습니다.

### 2.2 대시보드 고도화 (Dashboard)
- **희망 진로 코스 차트 추가**: 도제, 군특성화, 아우스빌둥 등 세부 교육 경로별 분포를 확인할 수 있는 차트를 신설했습니다.
  - 진학 희망자는 별도의 '진학' 카테고리로 분류하여 집계.
  - 데이터가 있는 항목만 범례에 표시하고, 지정된 정렬 순서(취업 관련 -> 진학 -> 제외인정자 관련)를 준수합니다.
- **레이아웃 재배치**: 
  - '취업 현황' 차트를 '기업 규모별 분포' 차트보다 앞에 배치하여 중요 지표의 우선순위를 조정했습니다.
  - 활용도가 낮은 '학반별 취업 현황' 차트를 제거하고 '자격증 현황' 차트의 너비를 확장했습니다.

### 2.3 모바일 UI/UX 최적화
- **카드 레이아웃 개선**: 학생 카드 상단에 `{학과} {반}반 {번호}번` 정보를 병기하고, 정보 그리드를 **3열 2행(총 6개)**으로 확장했습니다.
- **데이터 초기화 버그 수정**: 모바일 상세 모달에서 '선택 취소(비우기)' 시 `CLEARED` 문자열이 저장되던 문제를 해결하고 실제 `null`값이 DB에 반영되도록 수정했습니다.

### 2.4 학생 관리 페이지 기능 동기화
- **진로 코스 컬럼 추가**: 학급 관리 페이지의 핵심 항목인 '진로 코스'를 학생 관리 테이블에도 이식했습니다.
- **테이블 간소화**: '자격증' 컬럼을 제거하고 '비고'와 그룹 헤더를 통합하여 데이터 관리의 집중도를 높였습니다.
